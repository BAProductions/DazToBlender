## Prerequisites
---
* A compatible version of the [Daz Studio][DazStudioURL] application
  * Minimum: 4.10
* A compatible version of the [Blender][BlenderURL] application
  * Minimum: 2.80
* Operating System:
  * Windows 7 or newer
  * macOS 10.13 (High Sierra) or newer
